# ETH 2025 Seedling Distribution Dashboard

This dashboard is a Shiny application for visualizing and analyzing data from the ETH 2025 Decurrence Tree Value Survey. It provides interactive analytics on tree seedling distribution, farmer demographics, geographic trends, and phone ownership, all powered by real-time data from Google Sheets.

## Features
- **Live Data Loading**: Connects directly to a public Google Sheet for up-to-date survey data.
- **Interactive Filters**: Filter data by purpose and cluster.
- **Multiple Tabs**: Overview, Geographic Analysis, Demographics, Tree Species Analysis, Phone Ownership, and Help.
- **Modern UI**: Responsive design with custom theming and icons.
- **Error Handling**: Clear messages and retry options if data fails to load.
- **Visualizations**: Interactive charts and tables using Plotly and DT.
